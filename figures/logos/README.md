# Technology Logos for Chapter 3 (Implementation)

Place all technology logo PNG files in this folder:

- docs/figures/logos/

Use the exact filenames below (case-sensitive):

- vscode.png
- android-studio.png
- react-native.png
- expo.png
- nodejs.png
- supabase.png
- postgresql.png
- tailwind.png
- openai.png

## Recommended format

- File type: PNG
- Background: transparent if possible
- Recommended size: around 512x512 px (or larger, square preferred)
- Keep visual quality high and avoid blurry logos

## How it works in LaTeX

The report now checks each logo path automatically.

- If the file exists, the real logo is shown.
- If missing, a placeholder box is shown (compilation still succeeds).
