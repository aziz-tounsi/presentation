CHAPTER 1: GESTION DES DONNÉES

SLIDE 3
Title: Gestion des données
Content: La gestion des données est l'ensemble des opérations permettant de collecter, organiser, stocker, sécuriser et exploiter les données d'une organisation.

SLIDE 4
Title: Information et donnée
Content:
• Une donnée est un fait brut sans interprétation
• Une information est une donnée traitée qui possède une signification et peut être utilisée pour prendre une décision
• Exemple : "38" est une donnée, "La température est de 38°C" est une information

SLIDE 5
Title: Base de données
Content: Une base de données est un ensemble structuré de données stockées de manière organisée afin de faciliter leur consultation, leur modification et leur gestion.

SLIDE 6
Title: Les types de données
Content (with icons):

SLIDE 7
Title: BIG DATA
Content: Le Big Data désigne un ensemble très volumineux de données générées à grande vitesse et sous différents formats nécessitant des technologies spécifiques pour leur traitement. Exemples : Facebook, Google, YouTube, Amazon.
